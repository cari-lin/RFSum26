# Research Fellowship Summer 2026
## Graph-Based Demand Forecasting and Rebalancing Optimization in Urban Bike-Sharing Networks
By: Carina Lindfield

Advisor: Dr. Christopher Brown

### Main Question: How can network modeling and predictive algorithms be used to forecast station-level demand and optimize bike redistribution in urban bike-sharing systems?

All datasets used in this project are stored separately, due to most of them exceeding GitHub's file size limits. 
Download the datasets here: [Google Drive](https://drive.google.com/drive/folders/1D6hwYHJVnslupBQQX4o3CqG15dvN3-yy?usp=sharing)
/n After downloading, place the files in the file names ``data``. 

There is a miniature dataset stored directly in the ``data`` folder, which can be used to run a miniature version of the entire project. This data was collected randomly using `seed=42` for reproducibility.
