# information

## Purpose
This folder is for information, either visual, or where data was sourced.

## Contents
* `eda-trips.ipynb`: This notebook explores the main dataset. There are multiple graphs showing metrics like distribution of trip durations, total rides per month,
  ride density by week and hour, and a map of the locations of the bike stations.
* `info-indegoabout.ipynb`: This notebook is a copy/paste of what is found on the indego website (linked in the notebook). This is where the data format and data processing
was located. The link in this notebook also leads to the raw data used for the stations and trip datasets. 

## Inputs
* `eda-trips.ipynb`: This notebooks expects a file titled `trips.csv`.
* `info-indegoabout.ipynb`: This notebook does not expect any inputs
